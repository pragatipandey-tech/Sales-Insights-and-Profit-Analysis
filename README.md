# 📊 Sales Insights and Profit Analysis (Power BI)

## 📌 Overview

This project focuses on analyzing sales performance and profit trends using an interactive Power BI dashboard. It helps in identifying key business insights such as top-performing regions, product categories, and customer segments.

---

## 🎯 Objectives

* Analyze sales and profit performance
* Identify high and low performing regions
* Understand customer segmentation
* Track sales trends over time

---

## 📁 Dataset

* Superstore Dataset
* Source: https://github.com/plotly/datasets/blob/master/superstore.csv

---

## 🛠️ Tools & Technologies

* Microsoft Power BI
* Power Query
* DAX (Data Analysis Expressions)
